📌Project Overview
This project analyzes the impact of advertising budgets (TV, Radio, Newspaper) on Sales revenue. The objective is to identify high-performing marketing channels and build a predictive model to forecast sales.

🎯Business Problem
How do different advertising channels influence sales revenue, and which channel provides the highest return on investment (ROI)?

🛠Tools & Technologies
Python
Pandas
NumPy
Seaborn
Matplotlib
Scikit-learn (Linear Regression)

📊Exploratory Data Analysis (EDA)
Performed data cleaning and validation.
Conducted correlation analysis.
Identified trends between advertising spend and sales.
Visualized relationships using scatter plots and heatmaps.

🤖 Predictive Modeling
Built a Multiple Linear Regression model.
Achieved R² Score = 0.90, meaning 90% of sales variation is explained by advertising budgets.
Evaluated model performance using train-test split.

🔎 Key Insights
TV advertising shows the strongest positive impact on sales.
Radio advertising moderately influences revenue.
Newspaper advertising has minimal impact.
Marketing budget optimization can significantly increase ROI.

💡 Business Recommendations
Increase investment in TV campaigns.
Optimize Radio marketing strategy.
Re-evaluate Newspaper budget allocation.
Use predictive modeling for future sales forecasting.
